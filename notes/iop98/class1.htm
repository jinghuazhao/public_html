For this practical we use Dr Alison Mcdonald's data set on neuroticism.  The data will be on floppy disk in an SPSS data file called class1.sav.   Associated SPSS syntax file called class1.sps and output file called class1.lst are also provided. Create a subdirectory on your U: drive, and copy these files across.  Then run SPSS and read the data in by selecting the File menu and then Open Data file. Examine the variables on the speadsheet.  Variable id is the twin pair identifier, numbered consecutively from 1 to 522 for MZ pair, and from 523 to 794 for DZ pair. Variable n is the neuroticism score of the proband twin, and ncotwin the neuroticism score of the cotwin.  Note that each pair is entered as 2 observations, in which proband and cotwin status is reversed.
 

Analysis of Variance

We now perform an ANOVA on MZ and DZ twins in order to obtain test for (a) the presence of familial resemblance in MZ twins, (b) the presence of familial resemblance in DZ twins, (c) the presence of genetic effects.

Select Statistics, ANOVA models, Simple factorial, choose n as your Dependent variable, id as your Factor(s) and Define Range 1-522 for MZ pairs and 523-794 for DZ pairs.  Paste after correct selection to obtain SPSS syntax, which should be similar to the following.

* ANOVA for MZ.
ANOVA
  VARIABLES=n
  BY id(1 522)
  /MAXORDERS ALL
  /METHOD UNIQUE
  /FORMAT LABELS .

* ANOVA for DZ.
ANOVA
  VARIABLES=n
  BY id(523 794)
  /MAXORDERS ALL
  /METHOD UNIQUE
  /FORMAT LABELS .
 
These commands are in class1.sps. You can start from SPSS File menu and Open SPSS Syntax and call up this file into SPSS syntax window, use shift-cursor key to select part of the code for ANOVA (or Ctrl-A to block all the commands, but we do NOT recommend this now), then click execute button. Questions and Exercises:

(a) What is the test result for familial resemblance in MZ twins?
(b) What is the test result for familial resemblance in DZ twins?
(c) What is the test result for the presence of genetic effects?
(d) Estimate the intraclass correlation for MZ twins.
(e) Estimate the intraclass correlation for DZ twins.
(f) Do these intraclass correlations differ significantly from each other?

If you only have pen and paper, you can use the built-in calculator from Windows accessories.
 

Regression Analysis

It is possible to estimate the intraclass correlations using regression analysis.  Select  Transform, Compute wgt = 0.5,  Paste, and click execute button, then you should be able to see a new variable wgt in the data sheet. Now select Data, Weight cases, Weight cases by wgt .  For MZ and DZ twins separately, select Statistics, Regression, Linear, and choose ncotwin as dependent and n as independent variables, run the regression.  You can seperate select Data, Select Cases, If condition is satisfied, If..., zyg = 1.0 for MZ twins and  zyg = 0.5 for DZ twins. In order for the following analysis, we return to the mode that would use all the cases, which could be done by selecting Data, Select Cases, All cases.  The command and output are as follows

COMPUTE wgt =  0.5 .
EXECUTE .

WEIGHT
  BY wgt .

USE ALL.
COMPUTE filter_$=(zyg = 1.0).
VARIABLE LABEL filter_$ 'zyg = 1.0 (FILTER)'.
VALUE LABELS filter_$  0 'Not Selected' 1 'Selected'.
FORMAT filter_$ (f1.0).
FILTER BY filter_$.
EXECUTE .

REGRESSION
  /MISSING LISTWISE
  /STATISTICS COEFF OUTS R ANOVA
  /CRITERIA=PIN(.05) POUT(.10)
  /NOORIGIN
  /DEPENDENT ncotwin
  /METHOD=ENTER n.

USE ALL.
COMPUTE filter_$=(zyg = 0.5).
VARIABLE LABEL filter_$ 'zyg = 0.5 (FILTER)'.
VALUE LABELS filter_$  0 'Not Selected' 1 'Selected'.
FORMAT filter_$ (f1.0).
FILTER BY filter_$.
EXECUTE .

REGRESSION
  /MISSING LISTWISE
  /STATISTICS COEFF OUTS R ANOVA
  /CRITERIA=PIN(.05) POUT(.10)
  /NOORIGIN
  /DEPENDENT ncotwin
  /METHOD=ENTER n.

FILTER OFF.
USE ALL.
EXECUTE.

You should see that that the regression coefficients are the same as the intraclass class correlations estimated from ANOVA.  Now perform a DeFries Fulker analysis.  First calculate the overall mean, using the command.

DESCRIPTIVES
  VARIABLES=n
  /FORMAT=LABELS NOINDEX
  /STATISTICS=MEAN STDDEV MIN MAX
  /SORT=MEAN (A) .

The overall mean of n is m = 10.23.   Now compute mean-centred variables.

COMPUTE y = ncotwin - 10.23.
IF (zyg = 1) a = n - 10.23.
IF (zyg = 1) d = n - 10.23.
IF (zyg = 1) c = n - 10.23.

IF (zyg = 0.5) a = (n - 10.23)/2.
IF (zyg = 0.5) d = (n - 10.23)/4.
IF (zyg = 0.5) c = (n - 10.23).
EXECUTE .

We are now ready to do the actual regression analysis, with the following commands.

WEIGHT BY wgt.
REGRESSION
  /MISSING LISTWISE
  /STATISTICS COEFF OUTS R ANOVA
  /CRITERIA=PIN(.05) POUT(.10)
  /ORIGIN
  /DEPENDENT y
  /METHOD=BACKWARD a c.

REGRESSION
  /MISSING LISTWISE
  /STATISTICS COEFF OUTS R ANOVA
  /CRITERIA=PIN(.05) POUT(.10)
  /ORIGIN
  /DEPENDENT y
  /METHOD=ENTER a d.
